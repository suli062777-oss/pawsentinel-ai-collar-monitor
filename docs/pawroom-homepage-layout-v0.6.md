# PawRoom 首页布局修正 v0.6：上半状态场景，下半数据

日期：2026-07-07  
用途：修正 v0.5 首页仍不够美观、主次不够舒展的问题。参考“上半概览叙事 + 下半数据卡片”的产品页面结构，重新定义 PawRoom 首页布局。

---

## 1. 核心布局

首页采用上下分区：

- 上半部分：宠物当前状态 + 半幅家庭场景；
- 下半部分：与当前状态对应的数据卡片。

左侧全局任务栏保持不变。

---

## 2. 上半部分

上半部分是首页的第一视觉区域，应当承担“快速看懂宠物现在怎么样”的任务。

内容包括：

- 大标题：花花今日状态；
- 当前状态：一切正常 / 正在客厅休息；
- 状态标签：安全等级高、项圈已连接、电量 78%、当前区域客厅；
- 主要操作：进入桌面小世界；
- 次要操作：生成今日回放；
- 右侧或中右侧放置半幅像素家庭场景；
- 场景中展示宠物、客厅、沙发、宠物窝、饭盆、门口、路径点。

视觉原则：

- 上半区要有较大的留白；
- 状态文字要更大、更清晰；
- 场景不铺满全页，只作为“此刻状态”的可视化；
- 可以使用半圆弧、浅橙色背景形状或大面积柔和色块承托场景。

---

## 3. 下半部分

下半部分承载数据，不再抢占首页第一视觉。

推荐数据卡：

- 活动量 268 min；
- 休息 8.2 h；
- 心率 92 bpm；
- 呼吸 20 rpm；
- 区域停留；
- 数据置信度 72%；
- 最近提醒 / 今日时间线。

视觉原则：

- 采用横向卡片组或 1-2 行数据卡；
- 每张卡使用独立小图表；
- 卡片之间保持充足间距；
- 用橙、绿、蓝、紫区分指标，不要全部是灰白玻璃卡；
- 数据区服务于上半状态，而不是成为页面主视觉。

---

## 4. 必须保持

- 左侧任务栏保持不变；
- 无顶部全局栏；
- 无右侧抽屉栏；
- 宠物和家庭场景为 2D 像素风；
- UI 文本、按钮、卡片、图表为现代高保真风格；
- 页面尺寸为 1440 × 1024。

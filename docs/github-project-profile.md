## Naming Rules

- Final public name: `PawSentinel`
- Repository name: `pawsentinel-ai-collar-monitor`
- Earlier internal names: `PawRoom`, `PawGuard`
- Public README and GitHub About should use `PawSentinel`.
- Older design and code documents may keep `PawRoom` where renaming would create unnecessary implementation churn.

## Public Positioning

PawSentinel should be introduced as a soft hardware and software system for pet safety monitoring, not as a pure recording app or a medical device. The key promise is:

```text
Turn collar telemetry into safety states, gentle reminders, and a living desktop companion scene.
```

## Safety Boundary

Do not claim:

- medical diagnosis
- disease detection
- medical-grade vital monitoring
- replacement for veterinary care
- replacement for real surveillance video in high-risk situations

Use:

- safety reminders
- daily monitoring trends
- non-medical vital trend indicators
- approximate activity zone
- simulated telemetry for MVP validation

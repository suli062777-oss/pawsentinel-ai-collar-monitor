# PawRoom 高保真方向修正 v0.3：像素场景 + 液态玻璃 UI

日期：2026-07-07  
用途：修正 v0.2 “全界面像素化”过度的问题。v0.3 明确采用“像素宠物/场景 + PawRoom 液态玻璃 UI”。

---

## 1. 正确方向

PawRoom 高保真界面不是完整像素游戏 UI。

正确组合是：

> 主画布里的宠物、房间、家庭地图、今日小剧场使用 2D 像素风；界面框架、左侧栏、按钮、数据卡、设置表单继续使用 PawRoom 设计系统里的液态玻璃与现代高保真 UI。

---

## 2. 哪些必须像素化

必须像素化：

- 宠物 IP。
- 宠物状态贴纸。
- 家庭地图、房间、小世界场景。
- 今日历程里的小剧场、四格漫画、表情包预览。
- 电量页中的项圈/电池插图可以使用像素风插画。
- 添加宠物流程中的宠物分身预览。

---

## 3. 哪些不要像素化

不要像素化，保持液态玻璃/现代 UI：

- 左侧栏本体。
- 左侧栏按钮和高亮态。
- 数据卡、状态卡、提醒卡。
- 图表容器、设置表单、切换开关、按钮。
- 文字排版和数字。
- 主画布上的玻璃浮层。

文字必须清晰，不要做成低清像素字。

---

## 4. 左侧栏摆放规则

左侧栏参考用户示意图的是“位置和分组”，不是低保真灰块样式。

高保真表现：

- 一个竖向液态玻璃圆角栏，浅灰/半透明/柔和模糊。
- 内部按上下两块分区。

上半部分：主任务区，依次为：

1. 首页
2. 桌面小世界
3. 宠物数据
4. 今日历程
5. 设置

下半部分：动作/状态/宠物区，依次为：

1. 添加
2. 电量
3. 宠物块：小像素宠物头像 + 花花

高亮态：

- 当前入口使用液态玻璃胶囊或暖橙描边/底色。
- 添加、电量可以是圆形玻璃按钮。
- 宠物块是底部较大的液态玻璃圆角块。

---

## 5. 主画布规则

主画布仍是现代高保真产品界面：

- 背景为温暖奶油白。
- 主画布中放 2D 像素地图/房间/场景。
- 状态卡、数据卡、操作按钮作为液态玻璃浮层叠在像素场景上。
- 不使用右侧抽屉栏，不使用顶部全局栏。

---

## 6. 视觉关键词

推荐：

- liquid glass UI
- translucent rounded sidebar
- modern high fidelity dashboard
- pixel pet avatar
- 2D pixel home scene
- pixel world canvas
- glass floating status cards
- warm cream background
- orange smart collar
- readable Chinese UI

避免：

- full pixel UI
- pixelated text
- low fidelity grey sidebar
- 3D pet mascot
- photorealistic pet
- ecommerce layout
- hospital dashboard
- right sidebar
- top nav bar

---

## 7. 生成检查

每张图检查：

- 左侧栏是否是高保真液态玻璃，而不是低保真灰色占位。
- 左侧栏位置是否按上 5 + 下 3 分区。
- 宠物是否像素化。
- 主画布场景是否是 2D 像素风。
- 数据卡/按钮/文字是否保持现代高保真可读。
- 是否没有顶部栏和右侧栏。

# PawRoom 高保真页面候选 v10

## 生成边界

- 基准主色：`#FF8F3F`
- 左侧栏：保留固定竖向液态玻璃侧边栏，上半部分为：首页、桌面小世界、宠物数据、今日历程、设置；下半部分为：添加、电量、宠物信息。
- 像素化范围：宠物 IP、宠物所在场景、地图/房间/漫画等内容区域使用 2D 像素风；界面控件、卡片、按钮、数据图表保持液态玻璃高保真风格。
- 结构约束：三类页面各自保留原信息架构，不统一套用首页的橙色上半区结构。

# PawRoom image2 透明素材拆分清单 v0.2

## 用途

当前前端还原优先使用 DOM/CSS/SVG 复刻 UI 控件、卡片、文字、数据图表和交互状态。对于 CSS 难以高保真还原的像素宠物、房间场景、家具、道具、漫画画面和产品设备，则拆成 image2 透明底高清素材组件。

目标不是重新设计页面，而是提高对高保真稿中像素内容资产的还原度。

## 通用生成规则

- 输出透明 PNG/WebP。
- 不带文字。
- 不带 UI 卡片。
- 不带水印。
- 不要把阴影烘焙得过重，主要阴影由前端控制。
- 保持 2D pixel art 风格。
- 保持暖橙识别色 `#FF8F3F`。
- 宠物统一佩戴橙色智能项圈。
- 透视、光源、像素颗粒要与桌面小世界统一。

## image2 通用提示词模板

```text
生成一个 PawRoom 产品用的透明底 2D 像素风素材组件。
对象：[具体素材名称]
风格：温暖、可爱、精致的 2D pixel art，统一暖橙识别色 #FF8F3F。
场景：宠物安全看护与桌面宠物小世界。
要求：transparent background, isolated object, no text, no UI card, no watermark, crisp pixel edges, soft warm lighting, clean silhouette, consistent perspective.
输出：PNG/WebP，长边至少 1024px，边缘干净，适合叠加在液态玻璃 UI 和像素房间场景中。
```

## 验收标准

- 透明边缘干净，不出现白边或黑边。
- 不包含文字和 UI 控件。
- 可以在前端自由缩放和定位。
- 与 `#FF8F3F` 品牌橙和暖奶油背景协调。
- 替换素材后不破坏真实 DOM 文案、按钮和数据图表。
- 后续接真实数据时，只替换素材 URL 和状态数据，不重排页面。

# PawRoom 像素化宠物 IP 首次展示提示词 v0.1

日期：2026-07-07  
用途：首次使用 / onboarding 展示图中的宠物 IP 方向  
注意：只让宠物 IP 与首次展示小场景像素化，Web 界面、导航、卡片、表单等产品 UI 不像素化。

## 1. 统一风格规则

```text
Generate one highly distinctive pixel-art pet IP character for PawRoom first-use onboarding display.
Only the pet character and the small onboarding display scene should be pixel art.
Do not generate app UI, buttons, navigation bars, cards, forms, text, logos, watermarks, phone screens, desktop screens, or interface screenshots.
Use crisp pixel art, chunky dark pixel outline, clean flat color blocks, low noise, no grain, no painterly texture.
Make the pet strange, memorable, slightly weird, but still friendly and domestic.
Use cream white, pale mint, soft sky blue, pastel pink, warm orange PawRoom accent, and dark brown/near-black outline.
Avoid AI-generated look, realistic fur, glossy 3D, plush toy render, dense background, random symbols, extra limbs, distorted face, text, watermark.
```

## 2. 10 个生成方向

1. Cloud Guardian Puppy-Cat  
   方形云朵小狗猫混合体，软耳、困倦眼神、小橙色项圈信标。

2. Battery Corgi Loaf  
   电池形柯基面包，横向长身体、小短腿、卷尾、半闭眼。

3. Signal Cat Cube  
   信号猫方块体，猫耳、天线尾巴、像接收家中状态的小宠物。

4. Noodle Ferret Dog  
   长条面条雪貂狗，小方腿、路线尾巴、橙色项圈。

5. Mushroom-Ear Cat  
   蘑菇耳灰猫，大困倦眼、圆身体、奇怪但安静。

6. Doorway Guardian Bunny-Dog  
   门口守护兔狗，长方形耳朵像门柱，小钥匙尾巴。

7. Puffer Puppy Cloud  
   圆鼓鼓云朵小狗，小翅膀、小鼻子、像漂浮陪伴物。

8. Sprout-Tail Cat  
   发芽尾巴小猫，头顶小芽、植物尾巴、冷静好奇。

9. Pillow Seal Pet  
   枕头海豹宠物，猫耳、鳍状小爪、懒散斜眼。

10. Star-Nosed House Pet  
    星鼻小家宠，圆身体、三角耳、星形鼻子，奇怪但不吓人。

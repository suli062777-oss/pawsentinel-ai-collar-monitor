# PawRoom 高保真像素风设计补充 v0.2

日期：2026-07-07  
用途：覆盖 v0.1 高保真方向，将 PawRoom Web 高保真界面统一切换为“2D 像素宠物小世界”。

---

## 1. 本版核心变化

v0.2 不再使用软糖 3D、普通卡通、写实宠物或纯玻璃 dashboard 风格。

本版统一为：

> 像素化宠物 IP + 2D 像素风家庭/地图场景 + 现代可读 UI 面板。

要求：

- 所有宠物形象必须像素化。
- 所有宠物状态贴纸必须像素化。
- 所有家庭地图、房间、户外、小世界场景必须是 2D 像素风。
- UI 面板可以保持现代、清晰、圆角，但要带轻微像素游戏 UI 气质。
- 不做 3D、软糖、写实、照片、监控摄像头画面。

---

## 2. 视觉参考

参考用户提供的像素宠物 IP：

- 灰色像素猫，橙色项圈，蘑菇头饰。
- 奶油色像素猫，橙色项圈，叶子装饰。
- 奶油色像素小狗，橙色项圈，趴在蓝色垫子上。

参考用户提供的像素场景：

- 2D 像素房屋、森林、水边、木屋、暖光。
- 俯视或轻等距视角的像素地图。
- 高饱和但克制的像素块、清晰边缘、无写实纹理。

---

## 3. 左侧栏新规则

左侧栏参考用户最后一张图，采用一个单独的竖向圆角浅灰胶囊。

结构分为两大块：

### 上半部分：主任务区

从上到下依次为：

1. 首页
2. 桌面小世界
3. 宠物数据
4. 今日历程
5. 设置

表现：

- 文字在深灰小矩形/像素按钮内。
- 当前页面按钮高亮，可使用更深灰或暖橙描边。
- 按钮之间留出明显间距。

### 下半部分：全局动作与宠物区

从上到下依次为：

6. 添加
7. 电量
8. 宠物板块：显示像素宠物头像/小方块头像 + “花花”

表现：

- 添加、电量为小圆形/胶囊按钮。
- 宠物板块为底部较大的圆角块，像参考图中的底部宠物区域。
- 这里不叫“宠物信息”长标签，界面里可显示“花花”，但文档中仍对应“宠物信息”入口。

硬性规则：

- 不要顶部全局栏。
- 不要右侧抽屉栏。
- 不要右侧固定状态栏。
- 左侧栏不能被拆成多个栏。
- 左侧栏不是电商导航，不放购物车、搜索、品牌大 logo。

---

## 4. 主画布规则

主画布必须承载主要体验：

- 首页：2D 像素家庭地图 + 项圈数据浮层。
- 桌面小世界：完整像素房间/家园场景 + 像素宠物。
- 宠物数据：像素 UI 数据卡 + 小地图/像素图表。
- 今日历程：像素时间线 + 像素小剧场/四格漫画。
- 设置：现代设置卡片，但图标和装饰可像素化。
- 添加：像素宠物生成流程。
- 电量：像素项圈/电池状态中心。
- 宠物信息：像素宠物档案和素材库。

---

## 5. 风格关键词

推荐：

- 2D pixel art
- cozy pixel game UI
- top-down pixel home map
- isometric pixel room
- pixel pet avatar
- orange smart collar
- warm cream UI
- grey rounded left sidebar
- crisp Chinese UI text
- low anxiety monitoring
- modern readable dashboard panels

避免：

- 3D render
- claymorphism mascot
- realistic pet photo
- CCTV feed
- hospital dashboard
- ecommerce pet shop
- cyberpunk neon
- heavy glassmorphism
- random fantasy RPG combat HUD
- dense data tables

---

## 6. 生成检查清单

每张图检查：

- 宠物是否像素化。
- 场景是否为 2D 像素场景。
- 左侧栏是否分为上半主任务区、下半添加/电量/花花宠物区。
- 是否没有顶部全局栏。
- 是否没有右侧固定栏。
- 主画布是否最大。
- UI 是否仍然可读，不因为像素风牺牲文字清晰度。

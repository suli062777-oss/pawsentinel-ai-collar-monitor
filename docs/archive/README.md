# PawSentinel Archive

This folder stores useful process material that should not sit on the main public documentation path.

本目录用于保存仍有参考价值、但不适合直接放在公开评审路径上的过程材料。它们可以帮助后续复盘产品决策、设计生成、前端还原和用户证据采集，但不是 PawSentinel 当前提交包的主线内容。

## Folders

- `product-iterations/`: early concepts, old PRD versions, and planning notes.
- `design-process/`: information architecture, low-fidelity flows, visual exploration notes, and technical design studies.
- `frontend-restoration/`: frontend restoration plans and implementation logs.
- `media-prompts/`: image/video generation prompts and generation records.
- `research-process/`: evidence collection protocols and intermediate research notes.
- `evidence-scripts/`: helper scripts used during research and evidence processing.

## Asset Policy

Large exploratory assets were intentionally not archived here. Old screenshots, contact sheets, and unused generated image batches were removed from the current tree to keep the public repository lightweight and easier to review. If those files are ever needed again, they can still be recovered from Git history.

The public project path should start from [docs/README.md](../README.md), not from this archive.

## 1. 统一视觉原则

- 产品：PawRoom 宠物智能项圈。
- 核心能力暗示：GPS 定位、活动状态、生命状态趋势、PawRoom 软件连接。
- 形态：柔和胶囊主控舱 + 圆角项圈带 + 内侧柔性传感窗。
- 风格：温暖、简洁、亲肤、宠物日用品感，不像冷冰冰的监控设备。
- 材质：哑光微磨砂硅胶 / 柔软织物 / 素皮质感，低调金属触点。
- 细节：一颗柔和状态灯、低调 PawRoom 爪印压印、隐藏式磁吸充电触点、快拆扣。
- 画面：产品摄影或高级 3D 产品渲染，白/暖灰背景，柔光，干净阴影。
- 禁止：强科技灯带、巨大屏幕、复杂电路外露、夸张尖锐造型、儿童玩具感、医疗设备感。
